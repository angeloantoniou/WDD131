MovieVault Site Plan

01. Purpose

MovieVault helps users explore and save their favorite movies. It provides movie suggestions, lets users filter by genre, and save favorites to their personal list for later viewing.

02. Audience

This site is for movie lovers looking to discover, organize, and save must-watch films. Itâ€™s great for casual viewers, film students, or people who want curated lists of great movies.

03. Dynamic Elements

  • DOM interaction: Clickable movie cards reveal full descriptions in a modal popup.
  • Conditional logic: "Add to Favorites" button toggles to "Remove" if the movie is already saved.
  • Arrays: Movie data stored in an array and rendered using `.forEach()` and filtered with `.filter()`.
  • Modules: One module for movie data, another for UI handling and event listeners.

04. Logo

Logo created using Canva.

05. Colors and Fonts

  • Primary Color: #1B1B1B (Dark Gray/Black)
  • Accent Color: #FF3C38 (Bright Red)
  • Background: #F4F4F4 (Light Gray)
  • Font Headings: 'Bebas Neue', sans-serif
  • Font Body: 'Roboto', sans-serif

06. Content

  • Home Page: Welcome message, about section, top 5 trending movies.
  • Child Page (Movies): Grid of movie cards with poster, title, genre, year, and "Save" button.
  • Footer with contact info, social icons, and copyright.
